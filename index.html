v1.2.0 — What's New
Improved

RAF-Throttled Mouse Glow

Mouse glow and cursor tracking now runs through requestAnimationFrame — one DOM write per screen frame. Eliminates animation stutter entirely.

Improved

Removed CSS Transition Repaints

Removed transition: background from #glow and #aura — these were causing GPU-heavy repaints on every single mouse movement.

Improved

Spark Particle Cleanup

Mouse trail sparks now remove via animationend instead of setTimeout, preventing timer pile-up during fast movement.

Improved

Grid Animation Optimised

Background grid slowed to 32s cycle and promoted to its own GPU layer via will-change: background-position.

Fix

Landing Page Redirect

Fixed a bug where Get Started could bounce already-logged-in users directly to the dashboard, bypassing the login flow.
